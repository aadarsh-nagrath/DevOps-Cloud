## 📂 Repository Contents

The notes are categorized and structured as follows:

*   **☸️ Kubernetes & CKA**: Comprehensive preparation guides, resource specs, Kustomize/Helm configs, networking models, and k8s doubts resolver. See the [`Kubernetes/`](./Kubernetes) and [`CKA/`](./CKA) folders.
*   **🐳 Docker**: Cheat sheets for containers, optimization patterns, and advanced network configs in the [`docker/`](./docker) directory.
*   **📜 Scripting & Linux**: Bash scripting protocols, Linux command indexes, permissions references under [`scripting/`](./scripting).
*   **🛠️ Terraform & CloudFormation**: Infrastructure-as-code patterns, workspaces, environments, and testing models in the [`Terraform and CF/`](./Terraform%20and%20CF) directory.
*   **🚀 GitOps & CI/CD**: ArgoCD, Jenkins integration pipelines, GitOps best practices, and Knative serverless patterns.

## 🖥️ Notes Hub UI Viewer

We have added a custom, premium Single Page Application to browse, search, and view your notes in your browser with full syntax highlighting.

### Getting Started

1.  **Navigate to the Notes Hub directory**:
    ```bash
    cd notes-viewer
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Start the server**:
    ```bash
    PORT=3005 npm start
    ```

4.  **Open in browser**:
    Access [http://localhost:3005](http://localhost:3005) to read your notes dynamically!

### Features
*   **Sleek Dark Glassmorphism UI** with light/dark theme toggle.
*   **Sleek Technology Icons** corresponding to file context.
*   **Interactive Sidebar Tree Navigation** with real-time text filter.
*   **Code Syntax Highlighting** (YAML, Bash, Dockerfile, Terraform/HCL, JSON).
*   **Responsive layouts** suitable for side-by-side split screen reading while coding.
